WikiData query call


Using the WikiData query service to answer the following biological question:
"How many human genes have orthologs?"

See the GitHub repository MSB1015-Assignment-1 for the latest release and source code.



The amount of human genes containing orthologs:



Figure 1: Pie chart

This image indicates the amount of orthologs found in all human genes (containing orthologs). The proportions of the pie slices indicate which organisms has the most ortholog genes to human genes. This image cannot answer the 'amount of human genes', but rather the taxon mapping distribution.



Figure 2: Bubble chart

The Bubblegram shows the amount of orthologs per gene, indicative of several genes contraining many orthologs. The amount of bubbles is reduced to only contain genes with more than 6 orthologs available.
